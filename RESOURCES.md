# Learn to Claude — Resource Guide

A short, curated list of places learners can go for ready-made tools and
deeper reference. Written for the companion to draw from and for learners
to browse.

**Companion rules:** point at *one* resource when a debrief genuinely
calls for it (challenge files note where) — never a link dump, never
early, never somewhere that will overwhelm a beginner. Products move
fast: if a link has drifted, say so honestly and search for the current
home rather than guessing.

## Ready-made skills and plugins

- **Anthropic's skills repository** — github.com/anthropics/skills —
  open-source, ready-made skills (document creation, slides, spreadsheets
  and more) you can read, install, and borrow ideas from. *First pointer
  at Challenge 1.04, once the learner has built their own skill: "yours
  joins an ecosystem."* The domain series (2–5) each open by installing
  and later tailoring one of these.
- **Knowledge-work plugins** — Anthropic publishes open-source plugin
  collections for professional work (research, finance, marketing, legal,
  sales and more); search **"Anthropic knowledge work plugins"** or browse
  the plugin directory inside the product. *Relevant from the domain
  series onward, and a gold mine to tailor: pulling one apart and
  reshaping it to your own use case is a Series 7 move.*
- **The plugin/skill directory in the product** — Cowork and Claude Code
  both have a built-in way to browse and install plugins (in Claude Code:
  the `/plugin` command). The safe-adoption habits are taught in
  Challenge 66.

## Official reference

- **Claude help center** — support.claude.com — plain-language answers
  about Cowork, plans, settings, and data handling (the Challenge 9
  "check what your settings say" stop).
- **Claude Code documentation** — code.claude.com/docs — the full
  reference for skills, memory (CLAUDE.md), plugins, hooks, and settings.
  Becomes the learner's home reference from Series 7–9; before that,
  cite it rather than send them in.
- **Anthropic Academy** — anthropic.com/learn — Anthropic's own free
  courses, including the AI Fluency course built on the same 4D
  competency framework this curriculum uses.

## For the ambitious (later series)

- **Anthropic's cookbooks** — github.com/anthropics/claude-cookbooks —
  code-first recipes; developer-leaning, right audience by Series 9–10.
- **GitHub** — github.com — home of the Series 8 lessons; their first
  account arrives there, not before.

*Maintenance note: review these pointers each release — this page is the
only place in the curriculum where external links live, deliberately.*
